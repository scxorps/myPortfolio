DEMIO

A smart tool for predicting and organizing the fight against tuberculosis.

Demio is an AI-powered web app designed to help healthcare teams predict future TB outbreaks across regions and organize vaccination or awareness schedules efficiently. Built with machine learning (LSTM), geospatial data, and a practical planning system, Demio supports local health departments like SEMEP with real-time insights and planning tools.
